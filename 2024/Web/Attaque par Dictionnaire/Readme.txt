Titre : Attaque par bruteforce, dictionnaire ou exploitation des erreurs humaines
Catégorie : WEB
Équipe : WEB
Personnes : Ferreira Enzo

Description - Énoncé :

Lors de ce challenge l'utilisateur est face à une page web qui demande un mot de passe pour acceder au flag.


Solution :

il y a plusieurs mots de passe correctes présents dans le fichier mots_de_passe.txt qui peuvent être trouvés rapidement avec une attaque par bruteforce ou une attaque par dictionnaire. Ces mots de passe sont tous très simple et pourtant ce sont des mots de passe qui sont vraiment utilisés par de nombreux utilisateurs d'internet. 

L'utilisateur du challenge peut trouver le mot de passe par lui même pour acceder au flag, cela permet d'exploiter les erreurs humaines qui peuvent parfois être négligées dans la cybersécurité.

Si le challenger veut bruteforce le mot de passe il peut utiliser par exemple Hydra qui est un outil qui permet de bruteforce en utilisant un fichier qui sert de liste avec de nombreux mots de passe utilisés couramment trouvable sur internet.
