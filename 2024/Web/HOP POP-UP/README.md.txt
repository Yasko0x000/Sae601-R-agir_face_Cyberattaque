## Description - Énoncé :

Le Hop POP-UP Challenge est un défi de sécurité web qui met en lumière une vulnérabilité dans un formulaire interactif HTML et JavaScript. Explorer ce formulaire et identifier la faille de sécurité pour obtenir le flag.


# Solution :

1.Ouvrez le fichier index.html dans un navigateur web.

2.Remplissez les champs du formulaire et observez les données affichées en dessous.

3.Analysez le code JavaScript (script.js) pour comprendre la vulnérabilité.

4.Exploitez la vulnérabilité pour afficher le flag.
