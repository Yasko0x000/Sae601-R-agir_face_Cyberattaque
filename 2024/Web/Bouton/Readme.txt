Titre : Bouton désactivé
Catégorie : WEB
Équipe : WEB
Personnes : Ferreira Enzo

Description - Énoncé :

La page du challenge est une page simple avec des informations ainsi qu'un bouton désactivé.


Solution :

La chose importante se trouve en bas de la page c'est le bouton, il est désactivé et il est necessaire de l'activer pour obtenir le flag, pour cela il faut se rendre dans le code source et aller directement à la ligne du bouton, içi on peut changer le code et remplacer le "disabled" par "enabled" ensuite le bouton apparraitera en vert et deviendra cliquable pour afficher le flag.
