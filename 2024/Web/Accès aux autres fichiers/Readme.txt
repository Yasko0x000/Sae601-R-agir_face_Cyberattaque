Description - Énoncé :

La page du challenge est une page simple qui demande un identifiant et un mot de passe pour se connecter et accéder au flag.


Solution :

Pour trouver la solution il faut se rendre dans le code source, ici l'identifiant et le mot de passe ne sont pas visible directement mais on voit qu'un fichier json est utilisé pour vérifier les valeurs de l'utilisateur. On peut remplacer dans l'URL avec le nom du fichier pour y acceder, ainsi on verra le username et mot de passe correcte.
